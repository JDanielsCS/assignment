Technical experience:

.net experience

Dot net is a software framework originally designed to build apps for Windows machines, but has since been updated to work on many different platforms. Dot Net supports 32 programming languages, but the most common requirements for a Dot net developer are C#, F# and Visual Basic. I have not learnt any of these languages, and would start with C#. This Udemy course: https://www.udemy.com/course/c-and-net-core-for-beginners/ would be a great starting place, as it teaches C# and Dot net together.

Javascript

Javascript is a programming language used in web development. JavaScript is used alongside HTML and CSS to create interactive websites. Almost every developer job on Seek asks for JavaScript experience, because a lot of apps are turning to web based apps these days. I have not learnt JavaScript yet, however it is my top priority language to learn, and I am very excited for it. I will be using the following website to learn JavaScript: https://www.theodinproject.com/

React.js

As the .js suggests, React is a very popular JavaScript library, that has become commonplace in web development roles. It allows you to create interactive user interfaces for your web applications. I see this a lot on job boards, so this is a skill I would pick up after learning the fundamentals of JavaScript.
